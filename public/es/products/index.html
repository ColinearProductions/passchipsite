PASSCHIP Reader

Este lector de chip de control de acceso está especialmente diseñado para procesos avanzados de filtración de Bank Id a áreas de autoservicio o espacios de alta seguridad. La plataforma de seguridad está diseñada para un proceso de control de acceso seguro usando las tarjetas de acceso CHIP (tarjetas de crédito o simplemente tarjetas inteligentes) solo o combinado con PIN y tecnología biométrica.
