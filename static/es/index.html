Nuestros productos principales

Lector PASSCHIP

Este lector de chip de control de acceso está especialmente diseñado para procesos avanzados de filtración de Bank Id a áreas de autoservicio o espacios de alta seguridad. La plataforma de seguridad está diseñada para un proceso de control de acceso seguro usando las tarjetas de acceso CHIP (tarjetas de crédito o simplemente tarjetas inteligentes) solo o combinado con PIN y tecnología biométrica.

Software de gestión PASSCON

La administración Ethernet centralizada, utiliza las funciones únicas del software PASSCON para toda la red, ahorros importantes en poco tiempo y recursos específicos de los administradores de seguridad. Facilita la integración con CCTV, detección de incendios y de intrusos a través de cualquier plataforma de seguridad integrada a través de Wiegand, Ethernet y la interfaz serial.

Preguntas Frecuentes

Nuestro objetivo es estar enfocados en construir relaciones con nuestros clientes y la comunidad

¿Dónde se puede usar el sistema PASSCHIP?
El sistema PASSCHIP es ideal para bancos que necesitan una auto-entrada de 24 horas, para instituciones financieras, centros de datos, infraestructuras vitales y agencias militares y policiales
¿En qué se diferencia PASSCHIP de otros sistemas de seguridad?
PASSCHIP es el primer sistema de acceso del mundo para utilizar una gama de lectores para áreas de autoservicio que funcionarán con tarjetas de crédito con chip
¿Puedo integrarlo con nuestro sistema de control de acceso ya existente?
El lector PASSCHIP puede funcionar como un lector wiegand estándar de 26 bits
¿Qué tarjetas de crédito son compatibles?
Se admiten los tipos de tarjetas de crédito estándar (visa, mastercard, etc.) y se pueden agregar nuevos tipos de tarjetas de crédito según sea necesario
¿Puedo configurarlo con un horario?
PASSCHIP admite programación de tiempo definida por software, así como desencadenadores externos de hardware
¿Se puede personalizar el mensaje de bienvenida?
PASSCHIP admite todo el conjunto de caracteres Unicode - podemos mostrar mensajes en el alfabeto latino, árabe y chino

Testimonio

En 2013, nosotros y la empresa rumana independiente CONIC DESIGN sentamos las bases para una asociación sólida en nuestra actividad de seguridad y protección mediante el establecimiento de un acuerdo global para instalar las nuevas soluciones de control de acceso de CONIC DESIGN para asegurar toda nuestra red de agencias , en total más de 200 sucursales. La solución técnica consiste en instalar el sistema de entrada muy sofisticado PASSCHIP desarrollado por CONIC DESIGN para asegurar todas las entradas en las "zonas de banca privada" por los lectores de chip de tarjeta inteligente de IP Bank. El proyecto está en la mitad de la fase de implementación, y en esta etapa evaluamos que nuestros requisitos se cumplen completamente, incluso exceden nuestras expectativas iniciales solicitadas en la documentación de la licitación, el feedback de nuestros clientes es muy alentador. Nuestro mayor nivel de seguridad demuestra ahora que nuestra inversión está protegida por una de las plataformas de control de acceso más sólidas y estables que hayamos probado, que nos ofrece informes detallados y control total de IP sobre las transacciones de acceso público en nuestras agencias. Implementando PASSCHIP ofrecido por CONIC DESIGN, contamos con ahorros notables, reduciendo significativamente nuestros costos con guardias, transporte, departamento de atención al cliente y otros gastos relacionados, y al mismo tiempo mejoramos nuestra seguridad y la comodidad del cliente durante su actividad de autobanco en nuestras agencias. Recomendamos el producto PASSCHIP, que ofrece funciones avanzadas de control de acceso, gran flexibilidad, monitoreo de alarma instantánea y grabada, integración de seguridad de la información y sólidas funcionalidades de administración de IP.

- Manager de Seguridad del Banco
